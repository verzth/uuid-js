# uuid-js
UUID v4 for Javascript based on RFC4122 version 4

### Installation
#### NPM
```shell script
npm install --save @verzth/uuid-js
```

#### Yarn
```shell script
yarn add @verzth/uuid-js
```

### Implementation
```html
<head>
    <script src="{location}/uuid.min.js"></script>
</head>
```

### Usages
```javascript
    var id = uuid4()
```
